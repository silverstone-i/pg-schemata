## ğŸ”„ Migration Management

### New Features

- **`MigrationManager` Class**: Full-featured migration management with automatic tracking
- **`SchemaMigrations` Model**: Built-in model to track applied migrations
- **Bootstrap Utility**: Enhanced `bootstrap` function with PostgreSQL extension support
- **Migration Tutorial**: Complete step-by-step tutorial with working examples

### Migration Capabilities

- **Automatic Migration Tracking**: Tracks applied migrations in `schema_migrations` table
- **Transaction Safety**: All migrations run within transactions for rollback safety
- **Flexible Extension Management**: Configure PostgreSQL extensions during bootstrap
- **Migration Directory Support**: Organize migrations in dedicated directories
- **Status Reporting**: View applied vs pending migrations

### Code Example

```javascript
import { MigrationManager } from 'pg-schemata';

const manager = new MigrationManager({
  schema: 'public',
  dir: './migrations',
});

const { applied, files } = await manager.applyAll();
console.log(`Applied ${applied.length} migrations`);
```
